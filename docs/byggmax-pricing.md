# Byggmax Material Pricing Reference

Prices are approximate as of early 2026, sourced from byggmax.se.
All prices in SEK. Prices vary between stores and over time.

## How Materials Are Sold

| Category | Sold as | Unit |
|----------|---------|------|
| Lumber (reglar, virke) | Per running meter, declining lengths 2.7–5.4m | kr/m |
| Sauna panels (bastupanel) | Per running meter | kr/m |
| Exterior cladding (ytterpanel) | Per running meter, declining lengths 2.7–5.4m | kr/m |
| Roofing (takplåt TP20) | Per sheet, priced per m² | kr/m² |
| Insulation (mineralull) | Per package (~6–7 m²/pkg) | kr/m² |
| Membranes (vindskydd, ångspärr) | Per roll | kr/roll |
| Screws | Per pack (250 or 1000 pcs) | kr/pack |
| Bench boards (bastulav) | Per running meter | kr/m |
