# Building Standards Knowledge Base

Reference materials for building a freestanding sauna in Sweden, placed on an existing terrace.

## Scope

- **Building type:** Small freestanding sauna (< 15 m² floor area = attefallsåtgärd)
- **Location:** Southern Sweden (climate zone III, sk ≈ 1.5–2.0 kN/m²)
- **Foundation:** Sits on existing wood terrace (no concrete slab)
- **Heater:** Wood-burning kiuas (vedeldad bastuugn)
- **Roof:** Shed/monopitch, metal standing seam, 5° pitch

## Important notes

- All values are based on standards as of early 2025 (BBR 29, EKS 11). From 1 July 2025,
  Sweden's building regulations are being restructured — verify against the current edition.
- Manufacturer installation manuals (CE-marked) are legally binding for safety distances.
  Generic values in these docs are typical but do not replace product-specific documentation.
- A chimney installation (skorsten) requires an **anmälan** to the municipality even for
  attefallsbyggnader. The chimney sweep (sotaren) must inspect before first use.
- Electrical work in the sauna must be performed by a **registered electrical contractor**
  (behörig elinstallatör).
